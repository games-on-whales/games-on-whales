Wolf

An intelligent wolf is better than a foolish lion.
— Matshona Dhliwayo.

Rationale

The goal is to create an open source backend for Moonlight based on GStreamer.

In the process we would like to create a platform-agnostic C++ library that clearly defines the protocol and that can be used in other projects. While doing so, we are working on documenting and testing the protocol.

At the beginning it’ll be focused on running in Docker so that we can replace Sunshine in GOW but the code is already structured in order to support multiple platforms.
