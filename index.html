Games on Whales

Stream multiple virtual desktops and games running in Docker!

Features

Multi user

Allow multiple users to stream different content by sharing single remote host hardware.

Headless

On demand creation of virtual desktops with full support for any resolution/FPS without the need for a monitor or a dummy plug.

Virtual devices

Supports mouse, keyboard, touchpad and joypads with Gyro, Acceleration and touchpad support when using a DualSense.

Low latency

Implements the Moonlight protocol to stream with low latency to a wide variety of supported clients.

Linux and Docker first

Run games and apps in Docker containers, with support for Steam, Firefox, Retroarch, and more!

Open Source

Our projects are fully open source, MIT licensed, and we welcome contributions from the community.
